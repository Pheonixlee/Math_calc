# Math_calc
Calculation by coding 
Write a computer program to find the number of ordered pairs of prime numbers (p,q)(p, q)
(p,q) such that when

N=p2+q3N = p^2 + q^3N=p2+q3
is written in decimal (without leading zeros), each digit from 0 to 9 appears exactly once. For example, (109, 1163) is one such pair because 109^2 + 1163^3 = 1573049628.

Submit the numerical answer and the source code for the program.
